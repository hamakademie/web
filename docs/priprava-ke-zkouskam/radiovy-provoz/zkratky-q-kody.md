# Zkratky a Q-kódy používané v radioamatérské komunikaci

Od počátků rádiového provozu, kdy komunikace probíhala výhradně za použití telegrafie (morseovy abecedy), měli operátoři snahu zrychlit vysílání použitím zkratek pro některé často se opakující a ustálené fráze. 

Pro radiomatéry mají zkratky význam i v tom, že umožňují překonat jazykové bariéry, a vést konverzaci o běžných radioamatérských záležitostech bez ohledu na jazykové znalosti.

## Q-kódy

Nejznámější soustavou kódů jsou tzv. **Q-kódy charakterizované skupinou tří písmen, z nichž první je "Q"**.

::: info Info
Q-kódy jsou důvodem, proč bylo mezinárodně přijato pravidlo, které neumožňuje mít volací znak, jež má třípísmenný suffix začínající písmenem Q. 
:::

**V radioamatérské praxi se používá jen určitá část Q-kódů**.
Ačkoli mají Q-kódy původ v telegrafii a v telegrafii se tak používají nejvíce, některé z nich byly s příchodem hlasových (FONE) spojení přejaty i do provozu používající otevřenou řeč. 

**Některé Q-kódy se v praxi používají i ve zobecněném významu**, např. *QSO* znamená "radioamatérské spojení", *QTH* "umístění stanice", *QRO* "velký výkon", *QRP* "malý výkon" atd.

**Q-kódy mají dvě formy** - formu oznamovací a formu otázky, kde za Q-kódem následuje otazník. Přehled nejpoužívanějších je uveden dále v tabulce, ve skutečnosti jich však existuje mnohem více.

| Q kód | Význam | Význam otázky |
|-------|---------|----------------|
| QRM | Vaše vysílání je rušeno. | Je mé vysílání rušeno? |
| QRN | Vaše vysílání je rušeno atmosférickým rušením. | Je mé vysílání rušeno atmosférickým rušením? |
| QRO | Zvyšte výkon. | Mám zvýšit výkon? |
| QRZ | Volá vás ...... | Kdo mne volá? |
| QRX | Zavolám vás znovu v ...... UTC na kmitočtu ...... kHz; přeneseně: Čekejte. | Zavoláte mne znovu? přeneseně: Mám počkat? |
| QSB | Váš signál má únik. / Síla Vašich signálů kolísá. | Má můj signál únik? |
| QRS | Vysílejte pomaleji. | Mám vysílat pomaleji? |
| QRT | Přestaňte vysílat. | Mám přestat vysílat? |
| QRP | Snižte výkon. | Mám snížit výkon? |
| QRV | Jsem připraven. | Jste připraven? |
| QRU | Nic pro vás nemám. | Máte něco pro mne? |
| QSL | Potvrzuji příjem. | Můžete potvrdit příjem? |
| QRL | Jsem zaměstnán, nerušte. | Jste zaměstnán? přeneseně: Je tento kmitočet volný? |
| QTH | Moje stanoviště je ...... . | Jaké je vaše stanoviště? |
| QSO | Mohu pracovat přímo se stanicí ...... . | Můžete pracovat přímo se stanicí ......? |
| QSY | Přelaďte se na kmitočet ...... kHz. | Mám se přeladit na kmitočet ...... kHz? |
| QSV | Vyšlete sérii "VVV". | Mám vyslat sérii "VVV"? |
| QSK | Mohu vás slyšet mezi svými značkami. | Můžete mne slyšet mezi svými značkami? |
| QRJ | Nemohu vás přijímat, váš signál je příliš slabý. | Přijímáte mne špatně? |
| QRK | Čitelnost vašich signálů je ...... | Jaká je čitelnost mých signálů? |
| QSZ | Opakujte každé slovo dvakrát. | Mám opakovat každé slovo dvakrát? |
| QRA | Volací znak vaší stanice je ....... . | Jaký je volací znak vaší stanice? |
| QRG | Váš kmitočet je ...... kHz. | Jaký je můj kmitočet? |


## Zkratky

Zkratky byly a jsou další cestou ke zrychlení telegrafního provozu. Narozdíl od Q-kódů však nezkracují ustálené obraty a fráze, ale jednotlivá slova nebo slovní spojení. Zkratky ve většině případů vznikly z odpovídajících anglických slov.

Radioamatérských zkratek je veliké množství, proto je zde neuvádíme. Nastudovat je však můžete na odkazech s testovými otázkami, které jsou uvedeny níže.
